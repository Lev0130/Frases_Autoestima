# Frases Autoestima de Branden

Tras leer los 6 pilares de la Autoestima de Nathaniel Branden, quise llevar a cabo uno de los ejercicios que propone para analizate y llevar un registro de tu autoestima.

Este es el de las frases. Todos los días rellenar una serie de frases que propone el autor con lo primero qeu te venga a la cabeza, sin pensar demasiado.

Al realizar el ejercico un par de días, se me ocurrió automatizarlo para hacer el proceso más cómodo, y aquí presento el resultado para que cualquier usuario pueda utilizarlo.

Este programa se ha realizado usando argparse, por lo que para iniciarlo en la consola debenos nombrar a python y el archivo diario.py 


## Argumentos a nuestra disposición:

--start  (muestra las frases a rellenar)

-f1 (respuesta de la frase 1)

-f2 (respuesta de la frase 2)

-f3 (respuesta de la frase 3)

--buscar (busca las frases por fecha con el formato aaaa-mm-dd)
